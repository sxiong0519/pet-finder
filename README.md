## Getting Started

To get a local copy up and running follow these simple steps.

## Requirements

* VSCode
* Git
* NodeJS

## Installation 

* Clone the repository 

    `git clone git@github.com:sxiong0519/pet-finder.git`

* npm 

    `npm install`
    
* Run the code

    `npm start`
